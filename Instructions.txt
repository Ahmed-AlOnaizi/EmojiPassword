Overview
Flask (Python): Handles user registration, login, and database interactions (using SQLite).
React (JavaScript): Provides the user interface with three variants (A, B, C) for emoji-based PINs.
SQLite: Stores user credentials in a users.db database file (auto-created by SQLAlchemy).

Prerequisites
Python 3.7+
pip (or another Python package manager)
Node.js 14+ (includes npm)

Clone the Repository 
using: git clone https://github.com/Ahmed-AlOnaizi/EmojiPassword.git

Install Python dependencies by runnning this:
pip install -r requirements.txt


Navigate to the backend folder (where the app.py exists) and run:
python app.py

Open a new terminal without terminating the other one
Navigate to the frontend folder and run:
npm install
npm start

Debugging: If there is an error with a module try running:
npm install react-router-dom
